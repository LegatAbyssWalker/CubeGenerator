# CubeGenerator
A functional cube generator with states and events using OpenGL and C++.

# Dependancies
* OpenGL and GLEW.
* C++17 with a supporting compiler!

# Contains
* States system
* Shader and Mesh class
* Cube Generator class

# Bugs
* Camera is a little buggy and choppy.
